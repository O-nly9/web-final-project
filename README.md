# 非遗传承 - 中国非物质文化遗产文化传播平台

## 项目简介

本项目是一个展示中国非物质文化遗产的综合性网页平台，涵盖青花瓷、簪花、苗绣、皮影戏、昆曲、打铁花六大非遗项目。平台通过文字、图片、视频等多种形式，全方位展示非遗文化的起源、历史、技艺传承等内容，旨在传播中华民族优秀传统文化，让非遗文化走进现代生活。

## 技术栈

| 类别 | 技术 |
|------|------|
| 前端框架 | 原生 HTML5 + CSS3 + JavaScript（无框架依赖） |
| 样式 | 内联 CSS + 外联 styles.css |
| 图标 | Font Awesome 6.4.0 |
| 图片素材 | 本地 img 文件夹 + 视频 mp4 文件 |
| 数据存储 | LocalStorage（用于登录注册功能） |

## 运行方式

### 方式一：直接打开
双击 `index.html` 文件，使用浏览器打开即可运行。

### 方式二：本地服务器
```bash
# Python 3
python -m http.server 8080

# 或使用 VS Code Live Server 插件
```

然后访问 http://localhost:8080

## 功能模块

### 首页模块
- **视频轮播**：展示6种非遗文化宣传视频，自动轮播/手动切换
- **非遗卡片**：点击跳转各非遗详情页
- **搜索功能**：支持关键词搜索非遗项目

### 非遗详情页（共6个）
每个详情页包含：
- 英雄区大图横幅
- 项目简介/起源历史
- 发展历程时间线
- 经典剧目/作品展示
- 技艺特色/表演形式
- 传承保护现状
- 名家介绍
- 图片画廊（支持点击放大）

### 用户系统
- 用户登录/注册
- 表单验证
- LocalStorage 数据持久化
- 记住用户名功能

## 注意事项

1. 视频文件较大，首次加载可能较慢
2. 部分页面使用内联样式，请注意样式统一性
3. 登录注册数据存储在浏览器 LocalStorage 中
4. 图片放大功能支持鼠标滚轮缩放和拖拽移动
